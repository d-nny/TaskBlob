Powerful Features

Everything you need to run your own mail and web infrastructure

Powerful Mail Server

Complete mail server solution with Postfix, Dovecot, and PostgreSQL backend. DKIM, SPF, and DMARC support for enhanced security and deliverability.

Cloud DNS Management

Automated DNS configuration via Cloudflare API. Easily add domains, configure MX, DKIM, SPF, and other records. Automatic SSL certificate generation.

Web Admin Panel

Intuitive web interface for managing domains, email accounts, and server settings. Real-time monitoring, log viewing, and SSL certificate management.

Enhanced Security

Built-in security features including Fail2ban, automated SSL renewal, firewall configuration, and environment variable-based credential management.

Docker Containerization

Fully containerized setup with Docker and Docker Compose. Easily deploy, update, and scale your infrastructure. Isolated services for improved security.

Complete Email Solution

Full-featured email system with webmail interface, anti-spam protection, virus scanning, and mail forwarding. Support for multiple domains and users.

How It Works

Setting up your own mail server has never been easier

Clone the repository from GitHub and create your .env file from the template. Configure your domain, IP addresses, and API credentials.

Run the master setup script, which will guide you through setting up the various components. The script handles DNS configuration, SSL certificates, and mail server setup.

Access the admin panel at admin.yourdomain.com to manage your server. Add domains, create email accounts, monitor server status, and view logs.

Your mail server is now fully functional! Users can access their email via webmail.yourdomain.com or use standard email clients like Outlook, Thunderbird, or Gmail app.
